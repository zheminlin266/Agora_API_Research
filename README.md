# Fundamental Research

## Agora API Dashboards

- [Agora npm downloads dashboard](https://zheminlin266.github.io/Agora_API_Research/agora_npm_downloads_dashboard.html): 跟踪 Agora JavaScript、React、React Native、RTM 及 Agent 相关包的周度 npm 下载量。
- [Agora PyPI downloads dashboard](https://zheminlin266.github.io/Agora_API_Research/agora_pypi_weekly_downloads_dashboard.html): 跟踪 Agora Python 生态包的周度 PyPI 下载量，用于观察后端、token、测试和部署相关需求。
- [LiveKit npm downloads dashboard](https://zheminlin266.github.io/Agora_API_Research/livekit_npm_downloads_dashboard.html): 跟踪 LiveKit client、React、React Native 和 Agents 相关包的周度 npm 下载量。
- [LiveKit PyPI downloads dashboard](https://zheminlin266.github.io/Agora_API_Research/livekit_pypi_downloads_dashboard.html): 跟踪 LiveKit Python SDK、Server API、Agents 和插件包的周度 PyPI 下载量。
- [Twilio npm downloads dashboard](https://zheminlin266.github.io/Agora_API_Research/twilio_npm_downloads_dashboard.html): 跟踪 Twilio 音视频、语音、React Native 视频处理及通用 SDK 包的周度 npm 下载量。
- [Bandwidth npm downloads dashboard](https://zheminlin266.github.io/Agora_API_Research/bandwidth_npm_downloads_dashboard.html): 跟踪 Bandwidth RTC、WebRTC 和通用 SDK 包的周度 npm 下载量。
- [RTC competitor npm downloads dashboard](https://zheminlin266.github.io/Agora_API_Research/rtc_competitor_npm_downloads_dashboard.html): 跟踪腾讯 TRTC、ZEGO、阿里云 RTC、火山引擎 RTC 等竞品包的 npm 下载趋势。
