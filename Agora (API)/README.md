# Agora (API) Dashboards

- [Agora npm 周下载量看板](https://zheminlin266.github.io/fundamental_research/Agora%20%28API%29/agora_npm_downloads_dashboard.html): 跟踪 Agora JavaScript、React、React Native、RTM 和旧版 RTC 包的周度 npm 下载量，用于观察开发者需求信号。
- [Agora PyPI 周下载量看板](https://zheminlin266.github.io/fundamental_research/Agora%20%28API%29/agora_pypi_weekly_downloads_dashboard.html): 跟踪 Agora Python 相关包的周度 PyPI 下载量，用于观察后端、token server、测试和部署需求信号。
- [LiveKit npm 周下载量看板](https://zheminlin266.github.io/fundamental_research/Agora%20%28API%29/livekit_npm_downloads_dashboard.html): 跟踪 LiveKit client、React、React Native 和 Agents 包自发布以来的周度 npm 下载量，用于观察前端与实时通信开发者需求信号。
- [LiveKit PyPI 周下载量看板](https://zheminlin266.github.io/fundamental_research/Agora%20%28API%29/livekit_pypi_downloads_dashboard.html): 跟踪 LiveKit Python SDK、Server API、Agents 和 Plugins 包的周度 PyPI 下载量，用于观察 Python 生态需求信号。
- [Twilio npm 周下载量看板](https://zheminlin266.github.io/fundamental_research/Agora%20%28API%29/twilio_npm_downloads_dashboard.html): 跟踪 `twilio-video`、`@twilio/voice-sdk`、`@twilio/video-react-native-sdk`、`@twilio/video-processors` 和 `twilio` 的周度 npm 下载量，用于观察 Twilio 实时音视频及通用 SDK 的开发者需求变化。
- [Bandwidth npm 周下载量看板](https://zheminlin266.github.io/fundamental_research/Agora%20%28API%29/bandwidth_npm_downloads_dashboard.html): 跟踪 `bandwidth-rtc`、`@bandwidth/bw-webrtc-sdk` 和 `bandwidth-sdk` 的周度 npm 下载量，用于区分 Bandwidth RTC/WebRTC 包与通用 SDK 的开发者需求信号。
- [RTC 竞争对手 npm 周下载量看板](https://zheminlin266.github.io/fundamental_research/Agora%20%28API%29/rtc_competitor_npm_downloads_dashboard.html): 跟踪腾讯云 TRTC、ZEGO、阿里云 RTC 和火山引擎 RTC 选定 npm 包 2021 年以来的周度下载量。
