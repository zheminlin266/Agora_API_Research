# Agora (API) Dashboards

- [Agora npm weekly downloads dashboard](https://zheminlin266.github.io/fundamental_research/Agora%20%28API%29/agora_npm_downloads_dashboard.html): Tracks weekly npm downloads for Agora JavaScript, React, React Native, RTM, and legacy RTC packages to monitor developer demand signals.
- [Agora PyPI weekly downloads dashboard](https://zheminlin266.github.io/fundamental_research/Agora%20%28API%29/agora_pypi_weekly_downloads_dashboard.html): Tracks weekly PyPI downloads for Agora Python-related packages to monitor backend, token-server, and testing/deployment demand signals.
