Agora (API) Dashboards

  • Agora npm weekly downloads dashboard

    Tracks weekly npm downloads for Agora JavaScript, React, React Native, RTM, and legacy RTC packages to monitor developer demand signals.

  • Agora PyPI weekly downloads dashboard

    Tracks weekly PyPI downloads for Agora Python-related packages to monitor backend, token-server, and testing/deployment demand signals.

  • LiveKit npm weekly downloads dashboard

    Tracks weekly npm downloads for LiveKit client, React, React Native, and Agents packages from launch through the latest npm range data.

  • LiveKit PyPI weekly downloads dashboard

    Tracks weekly PyPI downloads for LiveKit Python SDK, Server API, Agents, and Plugins packages to monitor Python ecosystem demand signals.

  • Twilio npm weekly downloads dashboard

    Tracks weekly npm downloads for twilio-video, @twilio/voice-sdk, @twilio/video-react-native-sdk, @twilio/video-processors, and twilio.

  • Bandwidth npm weekly downloads dashboard

    Tracks weekly npm downloads for bandwidth-rtc, @bandwidth/bw-webrtc-sdk, and bandwidth-sdk.
